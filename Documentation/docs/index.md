## 🛠️ Configuration Requise

=== "Windows"
```shell
wsl --install -d Ubuntu sudo apt install python3 julia
```

=== "macOS"
```bash 
brew install python julia
```

=== "Linux"
```bash 
sudo apt update && sudo apt install python3 julia 
```
